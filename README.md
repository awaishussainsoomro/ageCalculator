# Age Calculator

A clean, browser-based age calculator. Select your date of birth, hit Calculate, and instantly see your exact age in years, months, and days.

## Features

- Calculates exact age in years, months, and days
- Prevents selection of future dates
- Handles edge cases like month-end and leap years
- Clean gradient UI with a responsive layout
- Zero dependencies | pure HTML, CSS, and JavaScript

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

No build tools or installation needed | this is a static frontend project.

1. Clone the repository
   ```bash
   git clone https://github.com/awaishussainsoomro/ageCalculator.git
   ```
2. Navigate into the project folder
   ```bash
   cd ageCalculator
   ```
3. Open `index.html` in your browser
   - Double-click the file, or
   - Use the **Live Server** extension in VS Code

## How It Works

1. Pick your date of birth from the date picker
2. Click **Calculate**
3. Your exact age appears instantly in years, months, and days
